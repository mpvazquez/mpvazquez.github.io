About Me

Hello

My name is Marco, and I'm a Front-End Web Developer based in New York City. I am currently working on an ecommerce product at Bloomingdales which is built on Backbone.js. The team’s focus is to build, test and scale MVP enhancements in a quick, iterative, and data-driven process on the company's browse and search pages. Prior to this, at Sprinklr I worked to build a React.js product for Fortune 500 enterprise companies which published fully-interactable client-facing web applications.

As far as technology goes, I enjoy working with powerful frameworks like React.js, but at the end of the day, I really love using simple, vanilla JavaScript. I will remove jQuery in lieu of using the DOM Web API every chance I get. Styling and CSS work is what I refer to as "my happy place." It’s extremely satisfying when I get to swap JS animations for better performant CSS ones.

My path into web development began at my first non-technical job out of college. On the first day at the job, I was given the “admin” username and password and was instructed to manage all the content on the company website. This was my first proverbial look behind the “magic” curtain of a website, where I tinkered with basic HTML and CSS. I'm so thankful for that opportunity because that job function and interest is what eventually led me to do what I love by building digital products today!
